## Portfolio Projects

### Premier League Home Advantage Analysis

**Description:** I wanted to see if home advantage still matters in the Premier League, specifically whether the fans at Old Trafford and Anfield actually make a difference. I used SQL in BigQuery to analyze 8 seasons of match data, looking at win rates, goal differences, and how home performance varies across teams and stadiums. The results were visualized in Tableau to make the findings easy to explore interactively.

**Repository:** [`premier-league-home-advantage`](https://github.com/Mahirrrr/premier-league-home-advantage)

**Skills:** SQL, BigQuery, Tableau, data cleaning, exploratory data analysis, data visualization

**Tools:** Google BigQuery, Tableau Public

### Last.fm Performativity Tracker

**Description:** I exported 4.5 years of my Last.fm listening history (152,000+ scrobbles) and built an 8-dimension scoring system in Python to measure how "performative" my music taste is. The concept came from Reddit discussions about people curating their libraries for social validation rather than genuine enjoyment. I defined distinct performativity archetypes — girl-appeal (Laufey, Keshi, Clairo), tough-guy rap (Playboi Carti, Future, Chief Keef), drain/internet-core (Bladee, Death Grips, JPEGMAFIA), and critical canon (Kendrick, Radiohead, Frank Ocean) — then built scoring functions around listening depth, trend chasing, discovery rates, and artist categorization. The whole analysis runs in a single Jupyter Notebook with 15+ visualizations. Final score: 46.8/100, mostly authentic.

**Repository:** [`Performativity-Tracker`](https://github.com/Mahirrrr/Performativity-Tracker)

**Skills:** Python, Pandas, NumPy, Matplotlib, Seaborn, data cleaning, feature engineering, exploratory data analysis, data visualization

**Tools:** VS Code, Jupyter Notebook
